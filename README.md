# AI Personal Tutor Dashboard (React + Tailwind + Vite)

A production-ready UI/UX codebase matching your Figma brief: Home, Lessons, Progress, Chat, Settings, and optional Tutor Analytics, plus a small component library and accessibility baked in.

## Quickstart
```bash
npm i
npm run dev
```

## Tech
- React + TypeScript + Vite
- Tailwind CSS (utility-first; responsive, accessible focus states)
- Recharts for analytics
- Framer Motion for subtle transitions
- Lucide icons

## Structure
- `src/pages/*` route-based screens
- `src/components/*` reusable UI
- `src/lib/mock.ts` demo data
- `src/styles.css` Tailwind layers and tokens

## Notes
- This simulates AI chat responses in `Chat.tsx` for prototype/demo.
- Replace mock data and add API integration as needed.
